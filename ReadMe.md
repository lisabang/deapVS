DEAP-VS
---

DEAP-VS is a wrapper around DEAP to allow DEAP to be used easily and quickly for the specific problem of variable selection.

| Dependencies:
| ----------------------------
| scikit-learn 0.16+*
| numpy 1.9+*
| scipy*
| pandas*
| ipython 3.0+*
| matplotlib*
| DEAP


asterisk (*) indicates that these packages are included in anaconda.
